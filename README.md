# In Your Service

A top-down pixel RPG where you play as Handyman the Knight, completing quests for three NPCs to prove your worth before a final confrontation with Sir Aldric.

## How to Run
1. Install [.NET 10 SDK](https://dotnet.microsoft.com/download)
2. Download FFmpeg full shared build from https://www.gyan.dev/ffmpeg/builds/
3. Extract and copy the DLLs into a folder called `ffmpeg/` in the project root
4. Run `dotnet run`

## Controls
- WASD / Arrow keys — Move
- E — Interact / Advance dialogue
- 1 / 2 — Dialogue choices
- Click — Attack (during combat)
- ESC — Exit
